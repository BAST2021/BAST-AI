# BAST-AI
BAST AI é uma plataforma de Machine Learning desenvolvida com Streamlit que permite importar datasets, treinar modelos de classificação e regressão, visualizar métricas e gerar previsões de forma simples e intuitiva.


## 🚀 Funcionalidades

- Upload de datasets CSV
- Classificação e Regressão
- Treinamento automático
- Visualização de métricas
- Matriz de confusão
- Gráficos interativos
- Download das previsões

## 🧠 Modelos suportados

### Classificação
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- SVM
- KNN
- Naive Bayes

### Regressão
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor
- SVR
- KNN Regressor

## 📦 Instalação

```bash
git clone https://github.com/SEU_USUARIO/ml-studio.git
cd ml-studio
```

Crie ambiente virtual:

```bash
python -m venv venv
```

Ative:

### Windows

```bash
venv\\Scripts\\activate
```

### Linux/Mac

```bash
source venv/bin/activate
```

Instale dependências:

```bash
pip install -r requirements.txt
```

---

## ▶️ Executar projeto

```bash
streamlit run model_stream.py
```
